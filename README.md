# ИИ-агент с интерпретатором кода
Реализован на OPEN AI с использованием tool = code_interpreter. Интерфейс сервиса реализовам на Streamlit как веб-страница.

Инструкции:
1. Зарегистрируйтесь на OPEN AI PLATFORM. Получите там API key (пользование ключом платное).
2. Склонируйте этот репозиторий себе на компьютер
3. Переименуйте файл ".env.example" в ".env"
4. Замените в этом файле ключ OPEN AI на свой. *Модель (MODEL_NAME) можно поменять на другую, если нужно.
5. Проверьте наличие всех импортируемых библиотек, с помощью команды `pip install` доустановите те, которых не хватает
6. Запустите с помощью `streamlit run web.py`

В папке \data найдете примеры ввода и вывода программы, а также 2 тестовых датасета (разные данные и размерность).
